Quick Start

  1. Go the dashboard
  2. Start one or more worker nodes
  3. Start Primary Server
  4. Get and Set some keys
  5. You can stop any node and see the redistribution of keys
  6. You can refresh and see the hashing ring for reference
  7. You can also see the primary server logs for reference
  8. Open this page and click "Go to Dashboard".
